Assignment name  : search_and_replace
Expected files   : search_and_replace.c
Allowed functions: write, exit
--------------------------------------------------------------------------------

Write a program that takes 3 arguments: a string, a character to search for,
and a character to replace it with. The program displays the string with all
occurrences of the search character replaced by the replacement character,
followed by a newline.

If the number of parameters is not 3, or if the search/replace parameters are
not single characters, simply display a newline.

Examples:

$> ./search_and_replace "Papache est un sabre" "a" "o" | cat -e
Popoche est un sobre$
$> ./search_and_replace "zaz" "art" "zul" | cat -e
$
$> ./search_and_replace "zaz" "r" "u" | cat -e
zaz$
$> ./search_and_replace "jacob" "a" "b" "c" "e" | cat -e
$
$> ./search_and_replace "ZoZ eT Dovid oiME le METol." "o" "a" | cat -e
ZaZ eT David aiME le METal.$
