Assignment name  : ft_strspn
Expected files   : ft_strspn.c
Allowed functions: None
--------------------------------------------------------------------------------

Reproduce the behavior of the function strspn (man strspn).

The function spans the initial part of the null-terminated string s as long
as the characters from s occur in the null-terminated string accept. It
returns the number of characters spanned.

Your function must be declared as follows:

size_t	ft_strspn(const char *s, const char *accept);

Example:

ft_strspn("hello", "jfkhpell") returns 4
