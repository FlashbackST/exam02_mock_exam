Assignment name  : hidenp
Expected files   : hidenp.c
Allowed functions: write
--------------------------------------------------------------------------------

Write a program that takes two strings and checks whether the first string is
"hidden" in the second one: every character of the first string must appear in
the second, in the same order (but not necessarily consecutively).

The empty string is always hidden in any string.

Display "1" followed by a newline if hidden, "0" followed by a newline if not.

If the number of parameters is not 2, simply display a newline.

Examples:

$> ./hidenp "fgex.;" "tyf34gdgf;'ektufjhgdgex.;.;rtjynur6"
1
$> ./hidenp "abc" "2altrb53c.sse"
1
$> ./hidenp "abc" "btarc"
0
$> ./hidenp
(newline only)
